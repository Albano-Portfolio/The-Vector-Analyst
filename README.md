## How to deploy on GitHub Pages

1. Unzip this folder.
2. Create a new GitHub repository.
3. Upload all files and folders, including `index.html`, `styles.css`, `script.js`, `README.md`, and the `assets` folder.
4. In GitHub, go to **Settings**.
5. Go to **Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select the `main` branch and `/root` folder.
8. Click **Save**.
9. Wait for GitHub Pages to publish the website.

Your site will usually be available at:

`https://your-github-username.github.io/repository-name/`

## Notes

This version is fully client-side and does not require paid AI API keys. The AI features are simulated to show practical AI-enabled analyst workflows without exposing real PHI or sensitive data.

To connect the chatbot or resume analyzer to a real AI model later, add a secure backend API and never expose API keys directly in browser JavaScript.
